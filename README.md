California Housing Price Prediction 

This project uses a neural network built with TensorFlow to predict house prices using the California Housing dataset.

Features
- Data preprocessing using StandardScaler
- Deep learning model using TensorFlow
- Evaluation using MAE
- Visualization of training results

Technologies Used
- Python
- TensorFlow
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

 Model Performance
- Loss: MSE
- Metric: MAE
